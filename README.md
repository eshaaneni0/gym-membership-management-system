# Gym Membership Management System

A desktop-based Gym Membership Management System developed using Java Swing and Object-Oriented Programming principles.

## Features

- Add Regular Members
- Add Premium Members
- Attendance Tracking
- Membership Upgrades
- Discount Calculation
- Loyalty Point System
- File Handling for Data Storage

## Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)
- File Handling

## OOP Concepts Implemented

- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
